## ğŸ› TROUBLESHOOTING

### **"Payment section not showing"**
- Clear browser cache (Ctrl+Shift+Delete)
- Refresh page (Ctrl+R)
- Try a different browser

### **"Images not loading"**
- Check internet connection
- Image URLs might be outdated
- Replace with your own images

### **"Cart empties on refresh"**
- This is normal! Data is stored in memory only
- To persist data, use localStorage or backend
- Add this to JavaScript:
```javascript
// Save cart to browser memory
localStorage.setItem('cart', JSON.stringify(cart));

// Load cart on page refresh
cart = JSON.parse(localStorage.getItem('cart')) || [];
```

### **"Login not working"**
- Check browser console (F12)
- Make sure JavaScript is enabled
- Clear cookies & cache
